# Cloudphoto Нургалеев Мансур 11-907

Для того чтобы запустить программу выполните следующие команды:
1) cd ~
2) git clone https://github.com/Mansur908/Vvot_Nurgaleev.git
3) export PATH=$PATH:~/Vvot_Nurgaleev/cloudphoto
4) chmod +x ~/Vvot_Nurgaleev/cloudphoto/
5) chmod +x ~/Vvot_Nurgaleev/cloudphoto/cloudphoto
6) pip install -r ~/Vvot_Nurgaleev/cloudphoto/requirements.txt
Выполните инициализацию:
7) cloudphoto init

Заполните запрашиваемые данные и программа будет доступна для использования.

Доступные команды:
 - cloudphoto upload — отправка фотографий в облачное хранилище.
 - cloudphoto download — загрузка фотографий из облачного хранилища.
 - cloudphoto list — вывод списка альбомов и фотографий в облачном хранилище.
 - cloudphoto delete — удаление альбомов и фотографий в облачном хранилище.
 - cloudphoto mksite — формирование и публикация веб-страниц фотоархива.
 - cloudphoto init — инициализация программы.
